This is FAQ for Wikia Activity Notifier (FrontEnd Edition).
  • Why this site is HTTPS but seems isn't?
  • Because this site sends a request to get Recent Changes on a Wiki, but in some Wikia's domains HTTP protocol is 80 (Simply HTTP) and not 443 (HTTPS) so we get HTTP(not S) responses. (Fixed on 1.2 version)
  • My Wikia/FANDOM account is used here?
  • No, definitely is unnecessary use Wikia's accounts here. We just need AJAX and done!
  • Is there an App/Client for Desktop?
  • Yes, you can found it here.
  • Is this site still under maintenance/developing?
  • Of course, you can see the version at top of the page.
  • Is this site open source?
  • Yes, we're hosting WAN's repository at GitLab
  • I detected bugs here, how can I report bugs?
  • You can reports bugs sending a mail to apps@saektide.com.
  • I want to log recent changes on my Discord server/Slack team, how can I do that?
  • In that case, WAL (WikiaActivityLogger) can help you. Credits to KockaAdmiralac.
