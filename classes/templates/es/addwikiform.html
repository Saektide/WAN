Solamente se requiere el interwiki del dominio. Por ejemplo: http://es.pokemon.wikia.com sería es.pokemon

El servicio requiere que la API del dominio esté en el directorio principal. Por ejemplo: la API de https://wt.saektide.com estaría ubicada en https://wt.saektide.com/api.php. Escriba solo el dominio, sin el protocolo HTTP.
