Acerca de Tsundoku

Tsundoku es una herramienta para monitorear cambios recientes sobre un sitio que tiene el software MediaWiki. Fue creado por Saektide (LemonSaektide en FANDOM).

¿Cómo funciona?

Tsundoku envia solicitudes POST (porqué? intentando con GET conseguimos una respuesta mas antigua) a la API del wiki de MediaWiki, por ejemplo: para Undertale Wiki (es.undertale.fandom.com) la url del API es "undertale.fandom.com/es/api.php". Con la acción "consulta" y la lista Cambios Recientes. Tsundoku no va a registrar cambios recientes de bots.
