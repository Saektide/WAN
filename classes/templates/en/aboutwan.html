About WAN

WAN is a tool for monitoring recent changes on a site that has MediaWiki software. It was created by Saektide (LemonSaektide at FANDOM).

How this work?

WAN sends a POST request (why? trying with GET we gets an older response) to site's MediaWiki API, example: for Undertale Wiki (at FANDOM) (undertale.wikia.com) the API url is "undertale.wikia.com/api.php". With the query action and the RecentChanges list. WAN won't reach bots' activity.
