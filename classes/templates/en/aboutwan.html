About Tsundoku

Tsundoku is a tool for monitoring recent changes on a site that has MediaWiki software. It was created by Saektide (Xakuzyo at FANDOM).

How this work?

Tsundoku sends a POST request (why? trying with GET we gets an older response) to site's MediaWiki API, example: for Undertale Wiki (at FANDOM) (undertale.fandom.com) the API url is "undertale.fandom.com/api.php". With the query action and the RecentChanges list. Tsundoku won't reach bots' activity.
