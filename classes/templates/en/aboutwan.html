About WAN

WAN is a tool for monitoring recent changes on a Wikia's wiki. It was created by Saektide (LemonSaektide at Wikia).

How this work?

WAN sends a POST request (why? trying with GET we gets an older response) to Wikia's wiki API, example: for Undertale Wiki (undertale.wikia.com) the API url is "undertale.wikia.com/api.php". With the query action and the RecentChanges list. WAN won't reach bots' activity.
